------------- | -------------

⌥⌘C| Extract Constant
⌥⌘V| Extract variable
 ⌘ /                            | (un)comment lines
⌥ ⌘ /                           |(un)comment block
⌘ P                          |   brings method parameters
^ H |inheritance hierarchy for a selected class 
⌘ , |           project & IDE settings
⌘ ; |          project structure
F1  |         Quick Documentation
⌥ Space |        Quick definition (of class / method etc.)
